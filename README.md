# linguabridgeLinguaBridge ist eine lokal gehostete Webanwendung (GitHub Codespace + Zugriff über localhost im Browser), die ausschließlich Deutsch ↔ Englisch in nahezu Echtzeit übersetzt. Beim Start verlangt die App Mikrofonzugriff und lässt den Nutzer ein Audio-Output-Device (z. B. Kopfhörer) auswählen. Das Frontend übernimmt nur Audioaufnahme und Wiedergabe, während im Backend möglichst viel über externe AI-APIs läuft: Erkennung, wann gesprochen wird (Pause/Turn-Taking), Live-Transkription, automatische Sprachrichtung DE→EN oder EN→DE, Übersetzung und Text-to-Speech. Sobald eine natürliche Sprechpause erkannt wird, wird der letzte Sprach-Chunk transkribiert, in die jeweils andere Sprache übersetzt und direkt in das passende Ohr des Kopfhörers ausgegeben. Beginnt die andere Person zu sprechen, wird die laufende Ausgabe unterbrochen und die Übersetzungsrichtung sofort gewechselt. Ziel ist eine minimalistische Demo, die eine flüssige deutsch-englische Unterhaltung ermöglicht – technisch simpel, aber mit klarer Orchestrierung im Hintergrund.
